# 🎸 Gerador de Progressões Harmônicas Ultra

Um gerador inteligente de progressões de acordes desenvolvido em HTML, CSS e JavaScript puro.

O objetivo do projeto é fornecer inspiração rápida para compositores, produtores, músicos e estudantes de harmonia, permitindo gerar progressões em diversos estilos musicais, escalas e modos, com recursos avançados de teoria musical.

### 🎵 Recursos Harmônicos Avançados

#### Inversões de acordes

Exemplos:

* C/E
* G/B
* Am/C

Permite gerar movimentos de baixo mais interessantes.

#### Empréstimo Harmônico

Introduz acordes emprestados do modo paralelo:

* iv menor
* ♭VI
* ♭VII

Exemplo:

C → Fm → C

#### Dominantes Secundários

Geração automática de dominantes secundários para criar maior tensão e resolução.

Exemplo:

A7 → Dm

## 🚀 Como usar

1. Abra o arquivo `index.html`
2. Escolha:

   * Estilo musical
   * Escala
   * Tom
   * Quantidade de acordes
3. Ative ou desative os recursos avançados
4. Clique em **Gerar Progressão**
5. Ouça, copie ou favorite o resultado
